Sipaa Operating System

Old skiftOS, relived.

What's that

SipaaOS is an operating system build with legacy skiftOS' base (and HJERT kernel)

and fixes all applications & components of skiftOS (like the paint app).

New apps & features are also included.

Pretty applications

SipaaOS includes neat applications like Paint, Text Editor, File Manager and more with

a beautiful UI.



Awesome for developers and advanced users

SipaaOS includes awesome APIs, editors and utilities to program!
